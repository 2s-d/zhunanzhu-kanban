我来详细讲解WebSocket服务器的工作模式和各种极端情况的处理方案。

WebSocket服务器的工作模式
WebSocket 是事件驱动的，不是轮询。服务器不会每时每刻去查数据库，而是"收到消息才处理"。

实际工作流程
前端 ──发送订阅消息──> 服务器
                      │
                      ▼
                收到消息立即处理
                （验证、查库、返回）
                      │
                      ▼
                处理完毕就等待
                （不占用资源）
各种极端情况的详细处理方案
情况1：前端先订阅，后端后推送（正常工作）
时间线：
1. 前端连接WebSocket
2. 前端发送订阅请求 { phone: '138xxxx', key: 'xxx' }
3. 服务器验证key正确
4. 服务器检查数据库，发现有历史数据 → 返回给前端
5. 服务器注册订阅，继续监听新数据
6. ...（等待）...
7. 后端推送新数据 { phone: '138xxxx', data: {...} }
8. 服务器推送给前端 ✓
结果：前端能收到历史数据 + 实时数据

情况2：后端先推送，前端后订阅（关键场景）
时间线：
1. 后端连接WebSocket
2. 后端推送数据 { phone: '138xxxx', data: {...} }
3. 服务器收到 → 存入数据库（持久化）
4. ...（前端还没订阅）...
5. ...（丢弃或等待）...
6. 前端连接WebSocket
7. 前端发送订阅请求 { phone: '138xxxx', key: 'xxx' }
8. 服务器验证key正确
9. 服务器检查数据库 → 找到历史数据
10. 服务器返回历史数据给前端 ✓
11. 服务器继续监听新数据
结果：前端订阅后能拿到历史数据 ✓

情况3：前端订阅了但key错误
时间线：
1. 前端发送订阅 { phone: '138xxxx', key: '错误key' }
2. 服务器验证key → 不通过
3. 服务器返回错误，不注册订阅
4. 后端推送数据 → 找不到订阅者 → 丢弃（或存库）
结果：key验证失败，不推送

情况4：前端断开连接
时间线：
1. 前端订阅成功
2. 前端突然断开（浏览器关闭/网络断开/刷新页面）
3. 服务器检测到close事件
4. 服务器删除订阅记录
5. 后端推送数据 → 找不到订阅者 → 丢弃（或存库）
结果：前端需要重新订阅

恢复方案：前端可以监听onclose事件，自动重连并重新订阅

情况5：后端（APP）断开连接
时间线：
1. 前端订阅成功
2. 后端断开（APP关闭/网络断开）
3. ...（等待）...
4. 后端重连 → 再次推送数据
5. 服务器推送给前端 ✓
结果：后端重连后继续工作

情况6：多个前端订阅同一个手机号
时间线：
1. 前端A订阅 { phone: '138xxxx', key: 'xxx' }
2. 前端B订阅 { phone: '138xxxx', key: 'xxx' }
3. 服务器存储：phone '138xxxx' → [前端A, 前端B]
4. 后端推送数据
5. 服务器遍历所有订阅者 → 都推送
结果：所有订阅同一手机号的前端都能收到

情况7：后端推送太频繁
问题：后端每秒推送100条，前端来不及处理

解决方案：
1. 前端做消息缓冲和批量处理
2. 服务器做限流（每秒最多推送X条）
3. 前端使用消息队列
情况8：服务器重启
问题：服务器重启后，内存中的订阅和连接都丢失

解决方案：
1. 订阅信息持久化到数据库
2. 服务器启动时从数据库恢复订阅
3. 后端重连后重新推送数据

// 内存缓存（加速读取）
let dataCache = loadFromDisk();

// ==================== 持久化函数 ====================
function loadFromDisk() {
  try {
    if (fs.existsSync(DB_FILE)) {
      const data = fs.readFileSync(DB_FILE, 'utf8');
      return JSON.parse(data);
    }
  } catch (e) {
    console.error('加载数据失败:', e);
  }
  return {};
}

function saveToDisk() {
  try {
    fs.writeFileSync(DB_FILE, JSON.stringify(dataCache, null, 2));
  } catch (e) {
    console.error('保存数据失败:', e);
  }
}

// ==================== 数据操作 ====================
// 存数据（覆盖式，只存最新）
function saveData(phone, data) {
  dataCache[phone] = {
    data: data,
    updatedAt: new Date().toISOString()
  };
  saveToDisk();
}

// 取数据
function getData(phone) {
  return dataCache[phone] || null;
}

// 取所有数据（用于管理员）
function getAllData() {
  return dataCache;
}

// 清理旧数据（每个手机号只保留最新1条）
function cleanupOldData() {
  const now = Date.now();
  const maxAge = 7 * 24 * 60 * 60 * 1000; // 7天
  
  for (const phone in dataCache) {
    const record = dataCache[phone];
    const age = now - new Date(record.updatedAt).getTime();
    
    if (age > maxAge) {
      delete dataCache[phone];
    }
  }
  
  saveToDisk();
  console.log('数据清理完成，当前记录数:', Object.keys(dataCache).length);
}

// 每天凌晨3点清理

const wss = new WebSocketServer({ port: WS_PORT });

console.log(`WebSocket服务器已启动: ws://0.0.0.0:${WS_PORT}`);

wss.on('connection', (ws, req) => {
  console.log('收到新的WebSocket连接');
  let deviceInfo = null;
  let isApp = false; // true=APP, false=前端
  
  ws.on('message', (message) => {
    try {
      const data = JSON.parse(message.toString());
      console.log('收到消息:', data.type);
      
      // ===== APP注册 =====
      if (data.type === 'register') {
        const { phone, key } = data;
        
        if (!phone || !key) {
          ws.send(JSON.stringify({ type: 'error', message: '缺少参数' }));
          return;
        }
        
        connections.set(phone, { ws, key, connectedAt: new Date().toISOString() });
        deviceInfo = phone;
        isApp = true;
        
        // APP连接时，推送最新数据给前端（如果有订阅）
        const record = getData(phone);
        const subscription = subscriptions.get(phone);
        if (subscription && record) {
          subscription.ws.send(JSON.stringify({
            type: 'realtime',
            data: record.data,
            updatedAt: record.updatedAt
          }));
        }
        
        ws.send(JSON.stringify({ type: 'registered', phone: phone }));
        console.log(`APP注册: ${phone}`);
      }
      
      // ===== 前端订阅 =====
      else if (data.type === 'subscribe') {
        const { phone, key } = data;
        
        if (!phone || !key) {
          ws.send(JSON.stringify({ type: 'error', message: '缺少参数' }));
          return;
        }
        
        // 验证key（从数据库或APP连接）
        const appConn = connections.get(phone);
        const dbRecord = getData(phone);
        
        const validKey = appConn?.key || dbRecord?.data?.key;
        if (key !== validKey) {
          ws.send(JSON.stringify({ type: 'error', message: '密钥错误' }));
          return;
        }
        
        // 保存订阅
        subscriptions.set(phone, { ws, key });
        deviceInfo = phone;
        isApp = false;
        
        // 返回历史数据
        const record = getData(phone);
        ws.send(JSON.stringify({
          type: 'subscribed',
          phone: phone,
          data: record?.data || null,
          updatedAt: record?.updatedAt || null
        }));
        
        console.log(`前端订阅: ${phone}, 有历史数据: ${!!record}`);
      }
      
      // ===== APP推送数据 =====
      else if (data.type === 'push') {
        const { phone, data: pushData } = data;
        
        // 存到数据库（覆盖）
        saveData(phone, pushData);
        
        // 推送给订阅的前端
        const subscription = subscriptions.get(phone);
        if (subscription) {
          subscription.ws.send(JSON.stringify({
            type: 'realtime',
            data: pushData,
            updatedAt: new Date().toISOString()
          }));
        }
        
        console.log(`APP数据推送: ${phone}`);
      }
      
      // ===== APP响应请求 =====
      else if (data.type === 'response') {
        handlePhoneResponse(data);
      }
      
    } catch (error) {
      console.error('解析消息失败:', error);
    }
  });
  
  ws.on('close', () => {
    if (deviceInfo) {
      if (isApp) {
        connections.delete(deviceInfo);
        console.log(`APP断开: ${deviceInfo}`);
      } else {
        subscriptions.delete(deviceInfo);
        console.log(`前端断开: ${deviceInfo}`);
      }
    }
  });
});

// ==================== 请求响应管理 ====================
const pendingRequests = new Map();

function sendRequestToPhone(ws, action) {
  return new Promise((resolve, reject) => {
    const requestId = Date.now().toString();
    const timeout = setTimeout(() => {
      pendingRequests.delete(requestId);
      reject(new Error('请求超时'));
    }, 10000);
    
    pendingRequests.set(requestId, { resolve, reject, timeout });
    ws.send(JSON.stringify({ type: 'request', requestId, action }));
  });
}

function handlePhoneResponse(data) {
  const { requestId, data: responseData, error } = data;
  const pending = pendingRequests.get(requestId);
  if (!pending) return;
  
  clearTimeout(pending.timeout);
  pendingRequests.delete(requestId);
  
  if (error) {
    pending.reject(new Error(error));
  } else {
    pending.resolve(responseData);
  }
}

/// 云同步服务 - 增强版
class CloudSyncService {
  static WebSocketChannel? _wsChannel;
  static String? _phone;
  static String? _key;
  static bool _isConnected = false;
  
  /// 初始化WebSocket连接
  static Future<bool> connect(String phone, String key) async {
    _phone = phone;
    _key = key;
    
    try {
      _wsChannel = WebSocketChannel.connect(Uri.parse(ServerConfig.wsUrl));
      
      await _wsChannel!.ready;
      _isConnected = true;
      
      // 发送注册消息
      _wsChannel!.sink.add(jsonEncode({
        'type': 'register',
        'phone': phone,
        'key': key,
      }));
      
      // 监听消息
      _wsChannel!.stream.listen((message) {
        _handleMessage(message);
      }, onError: (error) {
        print('[CloudSync] WebSocket错误: $error');
        _isConnected = false;
      }, onDone: () {
        print('[CloudSync] WebSocket连接关闭');
        _isConnected = false;
        // 自动重连
        _reconnect();
      });
      
      return true;
    } catch (e) {
      print('[CloudSync] 连接失败: $e');
      return false;
    }
  }
  
  /// 自动重连
  static Future<void> _reconnect() async {
    if (_phone != null && _key != null) {
      await Future.delayed(Duration(seconds: 3));
      await connect(_phone!, _key!);
    }
  }
  
  /// 处理收到的消息
  static void _handleMessage(dynamic message) {
    try {
      final data = jsonDecode(message as String);
      print('[CloudSync] 收到消息: ${data['type']}');
      
      if (data['type'] == 'request' && data['action'] == 'get_data') {
        // 服务器请求数据
        _sendCurrentData();
      }
    } catch (e) {
      print('[CloudSync] 处理消息失败: $e');
    }
  }
  
  /// 发送当前数据到服务器（主动推送）
  static Future<void> pushData(Map<String, dynamic> appData) async {
    if (!_isConnected || _wsChannel == null) {
      print('[CloudSync] 未连接，跳过推送');
      return;
    }
    
    try {
      _wsChannel!.sink.add(jsonEncode({
        'type': 'push',
        'phone': _phone,
        'data': appData,
      }));
      print('[CloudSync] 数据已推送');
    } catch (e) {
      print('[CloudSync] 推送失败: $e');
    }
  }
  
  /// 发送当前数据（响应服务器请求）
  static Future<void> _sendCurrentData() async {
    // 这里需要从DataRepository获取当前数据
    final data = DataRepository.instance.load();
    final appData = AppData();
    await data.then((d) {
      appData.schemaVersion = d.schemaVersion;
      appData.globalPointsTenths = d.globalPointsTenths;
      appData.projects = d.projects;
      appData.lastCheckInDate = d.lastCheckInDate;
      appData.lastFortuneDate = d.lastFortuneDate;
      appData.todayFortune = d.todayFortune;
      appData.themeSeedColorValue = d.themeSeedColorValue;
    });
    
    await pushData(appData.toMap());
  }
  
  /// 断开连接
  static void disconnect() {
    _wsChannel?.sink.close();
    _wsChannel = null;
    _isConnected = false;
  }
  
  /// 检查状态
  static bool get isConnected => _isConnected;
}

数据流：
1. APP数据变化 → 调用 CloudSyncService.pushData()
2. 服务器收到push → 存到data.json → 推送给前端
3. 前端订阅 → 服务器返回历史数据 → 继续监听实时数据

保证：
✓ 前端订阅 → 一定能拿到数据（从数据库）
✓ 数据库只存最新（覆盖式） + 定时清理（7天）
✓ APP主动推送 → 数据变化立即同步
